# 🎯 Guess the Number Game

A simple and interactive **Guess the Number** game built using **HTML, CSS, and JavaScript**. The game generates a random number and challenges the user to guess it, while tracking the number of attempts.

---

## 🚀 Features

- 🎲 Randomly generates a number between 1 and 100
- 💬 Gives hints — **Too High** or **Too Low**
- 🔢 Tracks the number of attempts
- 🏆 Congratulates the user on correct guess
- 🔄 Restart option to play again

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Structure of the game |
| CSS3 | Styling and layout |
| JavaScript | Game logic and interactivity |

---

```
javascript-guess-the-number/
├── index.html      # Main HTML file

```

---

## 🎮 How to Play

1. Open `index.html` in your browser
2. Enter a number between **1 and 100**
3. Click **Guess** button
4. Get hints — *Too High* or *Too Low*
5. Keep guessing until you find the correct number!
6. Click **Restart** to play again
